 # 问题
 ## 什么是IOC
 IOC是反转控制，类似于好莱坞原则，主要是依赖查找和依赖注入实现
 
 ## 依赖查找和依赖注入的区别
 依赖查找是主动或手动的依赖查找方式，通常需要依赖容器或标准API实现
 依赖注入则是手动或自动依赖绑定的方式，无需依赖特定的容器和API
 
 
 ## Spring 作为IoC容器有什么优势
 典型的IOC管理，依赖查找和依赖注入
 AOP抽象
 事物抽象
 事件机制
 SPI扩展
 强大的第三方整合
 易测试性
 更好的面向对象
